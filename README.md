# 🚗 Electric Vehicle Sales Analysis in India  

## Project Overview 
This project analyzes electric vehicle (EV) sales data across Indian states using Exploratory Data Analysis (EDA) and forecasting techniques.
 
## Objectives
- Analyze EV sales trends across states and years
- Identify top-performing states and EV categories
- Forecast future EV sales using time-series models

## Tools & Technologies 
- Python 
- Pandas, NumPy
- Matplotlib, Seaborn
- Statsmodels (Holt-Winters Forecasting)

## Key Insights
- EV adoption has increased rapidly in recent years.
- A small number of states dominate total EV sales.
- Two-wheelers are the most common EV type.
- Forecasting shows sustained growth in EV adoption.

## Future Scope
- State-wise forecasting
- Impact analysis of government subsidies
- Integration with charging infrastructure data
